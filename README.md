## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v8 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/endmr11/system_design.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run docs:dev
```
